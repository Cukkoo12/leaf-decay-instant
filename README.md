# \# Instant Leaf Decay

# 

# Leaves decay instantly when trees are cut. No more floating leaf blobs!

# 

# \## Features

# 

# \- Leaves decay the moment they lose connection to a log

# \- Fully configurable decay speed via `config/instant\_leaf\_decay.json`

# \- Particle and sound effects on decay (can be disabled)

# \- Works with all vanilla tree types

# \- Zero performance impact

# 

# \## Installation

# 

# 1\. Install \[Fabric Loader](https://fabricmc.net/use/)

# 2\. Install \[Fabric API](https://modrinth.com/mod/fabric-api)

# 3\. Download the latest release from \[CurseForge](https://www.curseforge.com/minecraft/mc-mods/leaf-decay-instant) or \[Modrinth](https://modrinth.com/)

# 4\. Drop the `.jar` file into your `mods` folder

# 

# \## Configuration

# 

# After first launch, a config file is created at `config/instant\_leaf\_decay.json`:

# 

# ```json

# {

# &#x20; "decayTicks": 2,

# &#x20; "particles": true,

# &#x20; "sound": true

# }

# ```

# 

# | Option | Type | Default | Description |

# |--------|------|---------|-------------|

# | `decayTicks` | int | `2` | Ticks to wait before leaves decay (0 = instant) |

# | `particles` | bool | `true` | Show particle effects when leaves decay |

# | `sound` | bool | `true` | Play sound effects when leaves decay |
